Smart Blind Stick using arduino.

1. Connect the arduino to your pc.
2. Paste this code in Arduino IDE.
3. Verify the code.
4. Connect all the tools as given in the circuit diagram. NOTE ---- CONNECT THE "TXD" AND "RXD" PINS (IN BLUETOOTH
    MODULE) ONLY AFTER UPLOADING THE CODE TO ARDUINO UNO NOT BEFORE THAT ELSE THE PROCESS WILL 
    FAIL.
5. Install the provided blutooth application in your mobile phone.
6. That's all.
